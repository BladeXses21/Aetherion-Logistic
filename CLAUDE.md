# Claude Code Settings

## Coding Guidelines

- **Коментарі та документація:** Обов'язково додавай гарне та зрозуміле пояснення **українською мовою** (docstrings/коментарі) до КОЖНОГО методу чи функції, які ти створюєш або змінюєш.
- **Складні та великі методи:** Для складних або великих методів з важливою бізнес-логікою завжди додавай розгорнуте пояснення того, ЩО і ЧОМУ робить метод. Вказуй, які аргументи очікуються та що метод повертає. Роби приклади використання, якщо це доречно.
- **Мова коментарів:** Усі нові коментарі в коді повинні бути виключно українською мовою.
- **Контроль версій (Git):** Після значних або важливих змін у коді завжди нагадуй мені зробити коміт (`git commit`) та, за необхідності, створити Pull Request (PR) для перегляду оновлень.
